Prompt 1: Tehtäväsi on toteuttaa yksinkertainen kokoushuoneiden varausrajapinta (API). Palvelun tulee tarjota käyttäjille seuraavat toiminnot: ● Varauksen luonti: Varaa huone tietylle aikavälille. ● Varauksen peruutus: Poista varaus. ● Varausten katselu: Listaa kaikki tietyn huoneen varaukset. Toimintalogiikka (business rules): ● Varaukset eivät saa mennä päällekkäin (kaksi henkilöä ei voi varata samaa huonetta samaan aikaan). ● Varaukset eivät voi sijoittua menneisyyteen. ● Aloitusajan täytyy olla ennen lopetusaikaa. Näiden ohjeiden lisäksi lisään vielä itse: Tee järjestelmään myös error handling ja tuota turvallista koodia. Toteutetaan tehtävä node.js ja TypeScript serverillä. Näin pystyn itse parantelemaan mikäli jotain huomaan Olisi myös hyvä ottaa huomioon, mikäli monta henkilöä käyttää API:a olisi hyvä ottaa huomioon mahdollinen threadaaminen tai muu vastaava queue menetelmä, ettei API hajoa, jos sitä kutsutaan monesta paikaa samaan aikaan.
